
Angaben zur Struktur 

1. 	Alles grafischen Einstellungen werden im CSS File gespeichert.
	HTML gibt nur die Struktur und ggf. die Aufteilung vor.

2. 	Klasse und Identifiertag bei HTML Bestandteilen werden IMMER zuerst angegeben.

3.	Bei Einbindungen wird die Quelle immer nach dem Klassen- oder Identifiertag geschrieben.
	Auf die Quellenangabe folgt der Alternativtext
	
Beispiel:

<img class="geiles_bild" id="geiles_bild" src="geiles_Bild.png" alt="Geiles Bild" height="100px">



Angaben zum Layout:

1.	Möglichst auch im jeweiligen CSS File

2.	Dynamisches Layout - Möglichst prozentuale Aufteilung oder in em
